                                Sample Description
================================================================================
功能描述：
此样例演示了时钟输出功能，可输出HSI波形。

Function descriptions:
This sample demonstrates the clock output function, which can output HSi waveforms.
================================================================================
测试环境：
测试用板：PY32F002B_STK
MDK版本： 5.28
IAR版本： 9.20

Test environment:
Test board: PY32F002B_STK
MDK Version: 5.28
IAR Version: 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 可由示波器观察到PA07输出HSI波形（24MHz）。

Example execution steps:
1. compile and download the program to MCU and run it;
2. The PA07 output HSI waveform (24MHz) can be observed by the oscilloscope.
